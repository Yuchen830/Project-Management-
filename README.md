# Project-Management-

## Description

This is a MERN Stack project designed with GraphQL, Apollo Client, and BootStrap UI. 
<br />
 
## Key Techniques

* **JavaScript/ HTML/ CSS**
* **MERN Stack: MongoDB Atlas Database/  Express.js/  React.js/  Node.js**
* **GraphQL / Apollo Client/ Bootstrap UI**
